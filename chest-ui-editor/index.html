What’s new in 2.0.0

Chest UI Editor has been updated. Highlights:

  • Tab - A tab button that allow switching between layouts using toggles
  • Dynamic grid - scrollable slots with configurable size, content height, and scrollbar textures
  • Button & close button - slot actions, custom textures, optional label/image; dismiss control with custom or no default art
  • Multiple Chest UIs - separate trigger, title, settings, and components per screen
  • Import ZIP - restore exported packs; validates formatVersion: 2
  • Per-slot textures - background, selected, and highlight art on containers and dynamic grid
  • Chest UI options - customizable background, size, and taggable title (use the label component for more control)
  • Double chest template - new vanilla-style double chest layout
  • Small chest template - now uses dynamic grid
  • Mobile - components, editor, preview, and templates tabs; touch placement and properties drawer
  • Layers & layout - layering fixes; new components placed on top; editable trigger for each UI
  • Export - generated UI organized into multiple files instead of one

Older browser saves and ZIPs without formatVersion: 2 (including version: "1.1.0") cannot be loaded. Save or export again with this build.
